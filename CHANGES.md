
JASP Release Notes
==================
 
Version 0.5
-----------

New Features:

 - Classical Linear Regression (basic implementation)
 - Bayesian ANCOVA
 - Bayesian Crosstabs
 - Custom models in Bayesian ANOVA
 - X-Squared in Classical Crosstabs

Changed:

 - Refinement of all the T-Tests (these are *really* polished now)

Fixed:

 - Fixes to add Linux support
 - Several bug fixes
